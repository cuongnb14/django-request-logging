Django Request Logging
===================

Install
=======

::

    pip install git+https://github.com/cuongnb14/django-request-logging.git@v1.0#egg=django-request-logging

Setting
=======

.. code:: python

    INSTALLED_APPS = [
        'request_logging',
        ...
    ]

Run migrate:

::

    python3 manage.py migrate
